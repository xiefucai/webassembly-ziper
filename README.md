# WebAssembly-Ziper

A WebAssembly implementation of JSZip with a fully compatible API, written in Rust.

## Features

- Drop-in JSZip API replacement
- Create, read, and modify ZIP archives in the browser
- DEFLATE compression support
- Path sanitization (zip-slip protection)
- Multiple output formats: `uint8array`, `arraybuffer`, `base64`, `string`, `array`
- Full TypeScript definitions generated by wasm-bindgen

## Quick Start

### Build

```bash
# Install wasm-pack (one-time)
cargo install wasm-pack

# Build the WASM package
wasm-pack build --target web --release
```

### Run the Demo

```bash
# Serve the demo directory
npx serve demo
# or
python -m http.server 8080 -d demo
```

Then open `http://localhost:8080` in your browser.

## API Compatibility

All major JSZip APIs are supported:

| JSZip API | Status |
|-----------|--------|
| `new JSZip()` | Supported |
| `zip.file(name)` | Supported |
| `zip.file(name, data [, options])` | Supported |
| `zip.folder(name)` | Supported |
| `zip.folder(regex)` | Supported |
| `zip.forEach(callback)` | Supported |
| `zip.filter(predicate)` | Supported |
| `zip.remove(name)` | Supported |
| `zip.generateAsync(options)` | Supported |
| `zip.loadAsync(data [, options])` | Supported |
| `zip.files` | Supported |
| `zip.comment` | Supported |
| `zipObject.async(type)` | Supported |
| `version()` | Supported |

## Usage Examples

### Create a ZIP

```javascript
import init, { JSZip } from './pkg/webassembly_ziper.js';

await init();

const zip = new JSZip();
zip.file("hello.txt", "Hello World\n");
zip.folder("images").file("readme.txt", "Image folder");

const content = await zip.generateAsync({ type: "uint8array" });
```

### Read a ZIP

```javascript
const zip = new JSZip();
const loadedZip = await zip.loadAsync(uint8ArrayData);

loadedZip.forEach((path, file) => {
    console.log(path, file.dir ? "[DIR]" : "[FILE]");
});
```

### Extract File Content

```javascript
const file = zip.file("hello.txt");
const text = await file.async("string");
const bytes = await file.async("uint8array");
const base64 = await file.async("base64");
```

### Compression Options

```javascript
zip.file("large.txt", repetitiveData, {
    compression: "DEFLATE",
    compressionOptions: { level: 9 }
});

const content = await zip.generateAsync({
    type: "uint8array",
    compression: "DEFLATE",
    compressionOptions: { level: 6 }
});
```
