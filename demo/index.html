WebAssembly-Ziper

A WebAssembly implementation of JSZip with compatible API

Loading WebAssembly module...

1. Create a Simple Zip

Add files and folders, then generate a zip archive.

const zip = new JSZip();
zip.file("hello.txt", "Hello World\n");
zip.folder("images").file("logo.png", imageData);
const content = await zip.generateAsync({type: "uint8array"});

3. Modify an Existing Zip

Load a zip, add files, remove files, and regenerate.

const zip = await new JSZip().loadAsync(zipData);
zip.file("new-file.txt", "New content");
zip.remove("old-file.txt");
const updated = await zip.generateAsync({type: "uint8array"});

4. Iterate and Filter Files

Use forEach and filter to work with zip contents.

zip.forEach((path, file) => console.log(path, file));
const txtFiles = zip.filter((path, file) => path.endsWith('.txt'));

Extract individual file contents from a zip.
